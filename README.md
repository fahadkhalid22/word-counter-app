# First Web App

My first web application, built as a learning project using the **Django** framework in Python.

## About
This project was built to learn the fundamentals of full-stack web development with Django — including the MVT (Model-View-Template) architecture, URL routing, and rendering dynamic HTML pages.

## Tech Stack
- Python
- Django
- SQLite (database)
- HTML (templates)

## How to Run
1. Install Python from python.org
2. Install Django: `pip install django`
3. Run database migrations: `python manage.py migrate`
4. Start the development server: `python manage.py runserver`
5. Open `http://127.0.0.1:8000/` in your browser
